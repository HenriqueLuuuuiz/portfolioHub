# 💻 Projetos Pessoais

Neste diretório guardo códigos e projetos independentes que desenvolvo para praticar minhas habilidades de programação fora do ambiente acadêmico.

### Projetos Atuais:
* **Gerador de Senhas (`gerador_senhas.py`):** Um script simples em Python que utiliza as bibliotecas `random` e `string` para gerar senhas fortes e seguras de tamanho customizável.
